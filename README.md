# Portofolio Pribadi Ananda Fahmuzna: TA-2-Praktikum-Pemrograman-Web

> Repositori Tugas Akhir Praktikum Pemrograman Web (TA Percobaan 2)

Proyek ini adalah website portofolio pribadi **statis** yang dibuat sebagai pemenuhan Tugas Akhir Judul 2 praktikum Pemrograman Web. Seluruh proyek ini dibangun menggunakan **HTML dan CSS murni** (tanpa framework, library, atau JavaScript).

Tujuan utama dari proyek ini adalah untuk mendemonstrasikan penguasaan dasar-dasar *web development* melalui implementasi layout yang **responsif** dan terstruktur.

## ‚ú® Fitur Utama

Website portofolio ini dirancang menggunakan struktur *multi-halaman* (multi-page structure), terdiri dari:

* **Multi-Halaman Terpisah:** Konten dibagi menjadi file `index.html`, `tentang.html`, `proyek.html`, dan `kontak.html`.
* **Struktur Dasar HTML5:** Penggunaan *semantic markup* yang rapi.
* **CSS3 & Flexbox:** Penataan layout dan styling dilakukan sepenuhnya oleh `style.css` menggunakan teknik Flexbox untuk desain responsif.
* **Navigasi Interaktif:** Terdapat *Navbar* dengan efek visual (seperti *hover* dan *transition*).
* **Komponen Profil:** Menyertakan aset gambar profil (`fotodiri.jpg`).
